## Creating some plots

Now we see that all penguins could be classified into three species, and they dwell three islands(not necessarily corresponded). First off, it might be helpful to create some plots demonstrating whether the features of each of these species corresponds with the islands or species.

In the following plot, I would like to see whether culmen length and culmen depth could together would show some pattern for different species.


```python
#relplot plot the relationship between two variables 
#the first two parameters are variables for relationship eval,
#the third one indicates the data to use, 
#and the last parameter 'hue' specifies which feature to use to cluster the penguins
sns.relplot("Culmen Length (mm)", "Culmen Depth (mm)", 
            data = penguins, hue = 'Island')
```

![blogpost0_16_2.png](/images/blogpost0_16_2.png)

Oops, it seems like culmen length and culmen depth could not separate out penguins on islands. This is to say, regardless of some patterns, there are penguins of for culmen length < 45mm and culmen depth > 16mm on each island.

Next, we check for the same thing, but cluster the penguins by species.


```python
sns.relplot("Culmen Length (mm)", "Culmen Depth (mm)", 
            data = penguins, hue = 'Species', size_norm = (5,5))
```


![blogpost0_16_2.png](/images/blogpost0_19_2.png)
    


Surprisingly, the above figure demonstrates that each of the species has its own cluster for culmen length and depth combination.
